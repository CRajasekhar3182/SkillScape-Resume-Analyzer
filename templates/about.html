BlueTick Resume Parser Application !!!

The main Aim of this application is to extract the important information from the resumes

  1. Machine Learning
  2. Natural Language Processing

Using NLP and ML Techniques we will extract the important information from the text and do necessary text cleaning and parse the information form the text.

Python Libraries used in Machine Learning

  • sklearn
  • Numpy
  • Pandas
  • Data Analysis

Python Libraries used in Natural Language Processing

  • SpaCy
  • Pandas
  • Regular Expression
  • String

As are combining two major technologies to develop the project, for the sake of easy to understand we divide the concepts into several stage of development.

Stage -1: We will setup the project by doing the necessary installations and requirements.

  • Install Python
  • Install Dependencies

Stage -2: We will do data preparation. That is used to clean the data for giving better Model performance.

  • Gather resume data in Text format
  • Overview on NLP pipeline
  • Extract Text from all Resumes
  • Clean and Prepare text

Stage -3: We will further clean the text and preprocess the data for train machine learning.

  • Prepare Training Data for ML Algorithms
  • Convert data into Array format using Embedding techniques

Stage -4: With the preprocess data we will train the ML Algorithms.

  • Configuring ML Model
  • Train the model

Stage -5: We will predict the resumes using NLP and model what we developed and create data pipeline for parsing text.

  • Load Model
  • collect the entire text from resumes
  • clean the data and use developed model for prediction
  • collect the results given by our model

Finally, we will put all together and create Resume Parser app.
